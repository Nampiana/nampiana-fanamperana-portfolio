Je m'appelle RAKOTOARIMANANA Santatriniaina Nampiana Fanamperana, j'ai 22 ans et j'habite à Madagascar, Ambohipo.

ChatSav

J'ai développé Chatsav, une application web et desktop dédiée au support technique, facilitant la communication entre techniciens et clients. Elle permet d'échanger via un chat pour résoudre les problèmes informatiques et offre un accès à distance si nécessaire, permettant ainsi de prendre le contrôle de l'ordinateur du client, à l’image de solutions comme AnyDesk ou VNC. Une version desktop est également disponible pour Windows. Cette application a été conçue avec Node.js, React.js, Electron et MongoDB.

e-CNaPS - Espace Pensionné

J'ai réaliser une application externe d'espace pensionner pour la Caisse National de Prévoyance Sociale. J'ai élaboré cette application de A à Z en utilisant Java Spring boot pour le backend, Angular pour le frontend et oracle comme système de gestion de base de données.C'est une application qui facilite la vie des pensionnés en leur évitant de se déplacer pour obtenir un certificat de vie, car cette démarche peut désormais être effectuée directement via l'application. Elle permet également aux pensionnés d'accéder à toutes leurs informations, comme leur bulletin de paie. De plus, un système de prise de photo est intégré pour attester de leur preuve de vie.
